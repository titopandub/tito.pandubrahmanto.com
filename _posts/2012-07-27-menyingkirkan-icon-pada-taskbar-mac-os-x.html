Karena suatu hal yang saya lupa, sebuah icon MobileMe Sync muncul di taskbar. Bingung bagaimana memindahkannya, karena opsi untuk menghilangkan icon tersebut tidak ada pada menu System Preferences. Hal ini disebabkan karena saya tidak memiliki account MobileMe (yang sekarang sudah ditutup).

Hari ini saya menemukan cara mudah untuk menghilangkan hal tersebut, caranya adalah tekan Cmd lalu klik kiri icon yang ingin kita singkirkan, lalu drag ke mana saja pada desktop dan lepaskan. Maka icon itu akan menghilang dari taskbar Anda.

Silahkan mencoba.
