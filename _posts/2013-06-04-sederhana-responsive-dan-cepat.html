Beberapa hari yang lalu saya merubah theme dari WordPress. Kali ini saya mencari theme yang sederhana, responsif dan cepat dalam proses loadingnya. Saya juga ingin memakai CSS dari Bootstrap untuk theme yang baru ini. Saya menemukan banyak theme seperti itu, dari yang gratis hingga berbayar. Namun akhirnya saya jatuh hati pada theme gratis dari 320press, yaitu WP-Bootstrap.

Theme ini sesuai dengan yang saya inginkan, sangat simple, responsive (cobalah untuk membuka ini di smartphone atau tablet) dan ringan. Memang dari sisi tampilan tidak begitu mewah, namun saya sangat puas. Kecepatan loading website menjadi jauh lebih cepat daripada memakai theme sebelumnya.

Karena memakai Bootstrap, saya jadi bisa merubah style dasar dari theme ini menggunakan theme Bootstrap di Bootswatch.com. Sehingga tampilannya tidak sama seperti style dasar dari Bootstrap. Berikut screenshot dari Theme saya saat ini (bisa berganti di kemudian hari).

Bagaimana pendapat para pembaca sekalian mengenai theme ini? Bagus atau tidak? Silahkan tinggalkan komentar ya.

Catatan: saya memakai theme Simplex dari Bootswatch.com
