Cita-cita saya sejak SMP adalah menjadi seorang Game Designer. Waktu itu saya sudah mencoba mempelajari Flash, Dark Basic, dll, tapi hasilnya kurang memuaskan. Entah karena saya belum mengerti mengenai konsep pemrograman (waktu itu), tidak pernah belajar dasar-dasar pemrograman sama sekali atau memang karena masih sibuk belajar. tapi sekarang saya mencoba kembali untuk mempelajari skill tersebut. Sebenarnya seluruh keputusan yang saya ambil sejak saat itu adalah untuk menjadi seorang Game Designer, tapi ya itu, jalannya berkelok-kelok. Saya harus jadi installer dan maintenance antena VSAT, jadi tukang oprek PC, jadi orang jaringan, buat aplikasi web, buat website, sampai sekarang akhirnya bisa juga mempelajari dan mengerti mengenai game development.

Saat saya belajar mengenai game development, saya langsung menetapkan tujuan (sesuai yang diperintahkan oleh buku yang saya baca) yaitu untuk membuat sebuah game yang sederhana. game yang terpikirkan saat itu ialah Congklak, dan sekarang akhirnya selesai juga pembuatan game congklak yang sederhana.

Fitur-fitur game ini terdiri dari dasar permainan congklak dan Easy AI. Namun menurut saya sudah cukup menyenangkan untuk memainkannya. Saya sudah memasang SWF nya di post ini, jadi bila ingin memainkannya silahkan saja. Saya akan sangat senang bila ada yang meninggalkan komentar dan masukan untuk game pertama saya. Masih banyak improvement yang bisa dilakukan dan saya sangat mengharapkan komentar dan saran dari para play tester.
